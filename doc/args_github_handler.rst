GitHub Handler command line
---------------------------

These are the parameters of :doc:`github_handler`. For GitHub Handler to work, these parameters are mandatory:

* ``--payload``
* ``--event``
* ``--trigger-url``
* ``--github-app-id``
* ``--github-private-key``
